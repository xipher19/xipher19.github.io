Audio Communication Units and Visual Ganglion Cells

In the past two weeks, I’ve focused on wrapping up the (4 different) ganglion cell implementations, to finish the first complete version of the digital retina. Now I’m implementing the audio CMU (see architecture above), particularly the digital versions of the cochlea and auditory nerve, to supplement visual stimulus processing.

Algorithms / Development

Push implementation and transportation

Direction selective ganglion cells (DSGC) implementation v2

  1. Greedily cluster SACs into groups of approximately 'sac_group_size' based on spatial proximity
  2. Integrate the input signal from many amacrine cells (in a similar centrifugal manner as SAC)
  3. Generate spikes based on the updated firing rates
Midget cells implementation v1

  1. Greedily cluster bipolar cells by spatial proximity
  2. Integrate input from r-g channel in cone bipolar cells, with spatial weighting, as they supporting high-acuity vision and red–green color opponency
  3. Generate spikes based on the stimulus
Parasol cells implementation v1

  1. Greedily cluster bipolar cells by spatial proximity
  2. Integrate input from rod / motion channel and cone brightness channel, with spatial weighting
  3. Generate spikes based on the stimulus
Small bistratified cells implementation v1

  1. Greedily cluster bipolar cells by spatial proximity
  2. Integrate input from b-y channel in cone bipolar cells, with spatial weighting
  3. Generate spikes based on the stimulus
